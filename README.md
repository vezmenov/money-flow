# Money Flow

Базовый каркас приложения на Angular + Vite.

Документация по UI/UX и архитектуре: `Docs/README.md`.

## Быстрый старт

```bash
npm install
npm run dev
```

## Скрипты

- `npm run dev` — локальная разработка
- `npm run build` — сборка
- `npm run preview` — локальный просмотр сборки
- `npm run ui:snap` — детерминированные UI-скриншоты (см. `Docs/Snapshots.md`)

## План разработки

- Роутинг и базовые экраны.
- Форма добавления расхода на главной.
- Конфигуратор категорий.
- Дашборды (обзор и аналитика).

Краткие следующие шаги: настроить хранилище данных, добавить экспорт/импорт, расширить аналитику.
