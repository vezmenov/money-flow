# Docs

Оглавление по изменениям UI/UX и архитектуре.

- `Docs/LunaGlass.md` — визуальный код (токены, материалы, состояния, do/don't).
- `Docs/Responsive.md` — адаптив: `>=700px` сайдбар, `<700px` bottom bar, safe-area, FAB.
- `Docs/UI-Kit.md` — shared UI-kit (кнопки/инпуты/модалка/иконки/календарь), правила использования.
- `Docs/Dashboard.md` — дашборд трат: метрики, агрегации, интерактив.
- `Docs/Transactions.md` — быстрое добавление траты (FAB + модалка).
- `Docs/Snapshots.md` — детерминированные UI-скриншоты через Playwright (для итераций по внешке).
- `Docs/DevLog.md` — краткий лог работ.
